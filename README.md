# Phone Mobile Store

This project is a simple and modern HTML landing page for a smartphone store called Lucas Mobile.

## What it includes
- A stylish homepage layout
- A hero section for featured phones
- Navigation and call-to-action buttons
- A clean purple-themed design

## Purpose
This page is designed to showcase a mobile store website in a polished and attractive way.

## Supabase backend setup
1. Create a Supabase project at https://supabase.com.
2. Run the SQL from supabase-schema.sql in the SQL editor.
3. Open supabase-config.js and replace the placeholder URL and anon key with your project values.
4. Open the page and submit the reservation form. Entries will be stored in the reservations table.
